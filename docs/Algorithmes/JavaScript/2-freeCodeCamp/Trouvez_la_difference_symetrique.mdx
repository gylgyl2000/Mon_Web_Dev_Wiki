# Trouvez la différence symétrique

[Learn Algorithms: Find the Symmetric Difference | freeCodeCamp.org](https://www.freecodecamp.org/learn/coding-interview-prep/algorithms/find-the-symmetric-difference)

<div className="algorithmes">

Le terme mathématique _différence symétrique_ (`△` or `⊕`) de deux ensembles est l'ensemble des éléments qui sont dans l'un des deux ensembles
mais pas dans les deux. Par exemple, pour les ensembles `A = {1, 2, 3}` et `B = {2, 3, 4}`, `A △ B = {1, 4}`.

La différence symétrique est une opération binaire, ce qui signifie qu'elle n'opère que sur deux éléments. Ainsi, pour évaluer une expression
impliquant des différences symétriques entre _trois_ éléments (`A △ B △ C`), vous devez effectuer une opération à la fois. Ainsi, pour les
ensembles `A` et `B` ci-dessus, et `C = {2, 3}`, `A △ B △ C = (A △ B) △ C = {1, 4} △ {2, 3} = {1, 2, 3, 4}`.

:::info Instructions
Créez une fonction qui prend au moins deux tableaux et renvoie un tableau de leur différence symétrique. Le tableau renvoyé doit contenir
uniquement des valeurs uniques (_pas de doublons_).
:::

:::tip missions
`sym([1, 2, 3], [5, 2, 1, 4])` doit renvoyer `[3, 4, 5]`.<br />
`sym([1, 2, 3], [5, 2, 1, 4])` ne doit contenir que trois éléments.<br />
`sym([1, 2, 3, 3], [5, 2, 1, 4])` doit renvoyer `[3, 4, 5]`.<br />
`sym([1, 2, 3, 3], [5, 2, 1, 4])` ne doit contenir que trois éléments.<br />
`sym([1, 2, 3], [5, 2, 1, 4, 5])` doit renvoyer `[3, 4, 5]`.<br />
`sym([1, 2, 3], [5, 2, 1, 4, 5])` ne doit contenir que trois éléments.<br />
`sym([1, 2, 5], [2, 3, 5], [3, 4, 5])` doit renvoyer `[1, 4, 5]`.<br />
`sym([1, 2, 5], [2, 3, 5], [3, 4, 5])` ne doit contenir que trois éléments.<br />
`sym([1, 1, 2, 5], [2, 2, 3, 5], [3, 4, 5, 5])` doit renvoyer `[1, 4, 5]`.<br />
`sym([1, 1, 2, 5], [2, 2, 3, 5], [3, 4, 5, 5])` ne doit contenir que trois éléments.<br />
`sym([3, 3, 3, 2, 5], [2, 1, 5, 7], [3, 4, 6, 6], [1, 2, 3])` doit renvoyer `[2, 3, 4 , 6, 7]`.<br />
`sym([3, 3, 3, 2, 5], [2, 1, 5, 7], [3, 4, 6, 6], [1, 2, 3])` ne doit contenir que cinq éléments.<br />
`sym([3, 3, 3, 2, 5], [2, 1, 5, 7], [3, 4, 6, 6], [1, 2, 3], [5, 3, 9, 8 ], [1])` doit renvoyer `[1, 2, 4, 5, 6, 7, 8, 9]`.<br />
`sym([3, 3, 3, 2, 5], [2, 1, 5, 7], [3, 4, 6, 6], [1, 2, 3], [5, 3, 9, 8 ], [1])` ne doit contenir que huit éléments.
:::
