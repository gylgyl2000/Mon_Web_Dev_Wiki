# Introduction du cours

[Découvrez les métiers de développeur - OpenClassrooms](https://openclassrooms.com/fr/courses/6817086-decouvrez-les-metiers-de-developpeur)

<Vimeo vimeoId="451416683" />

<br />

Vous avez entendu parler du métier de développeur mais vous ne comprenez pas vraiment de quoi il s'agit ? Le code, la programmation et les
méthodes agiles ne font pas de sens pour vous ? Dans ce cours, nous reprenons tout depuis le début.

Nous verrons ce qu'est un **développeur**, ou un programmeur, et son quotidien en entreprise. Dans la deuxième partie, nous apprendrons
à différencier les métiers de développeur et leurs interactions, puis nous zoomerons sur **chaque spécialité** et ses compétences. Dans
la dernière partie, vous verrez comment vous former en tant que développeur, les différentes manières d'exercer le métier et les
perspectives **d'évolution**.

Alors, prêt à rentrer dans le monde des codeurs ? Suivez-moi dans ce cours !

:::info
À l'issue de ce cours, vous serez capable de :
- situer le métier de développeur dans l'entreprise ;
- différencier les responsabilités de chaque spécialité de développeur ;
- évoluer en tant que développeur.
:::

:::caution !
**Prérequis** : aucun.
:::
