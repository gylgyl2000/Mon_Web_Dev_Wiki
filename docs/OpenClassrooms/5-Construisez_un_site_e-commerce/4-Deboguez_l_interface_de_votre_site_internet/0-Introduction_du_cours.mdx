# Introduction du cours

[Déboguez l’interface de votre site internet - OpenClassrooms](https://openclassrooms.com/fr/courses/7159296-deboguez-l-interface-de-votre-site-internet)

<Vimeo vimeoId="565510389" />

<br />

Vous aimeriez résoudre vos bugs de manière efficace ?

Dans ce cours, vous allez découvrir les techniques qui vont vous permettre de déboguer votre site de manière efficace.

Vous apprendrez à identifier le type de bug puis à le résoudre. Que ce soit un bug HTML, CSS, sur le DOM ou l’API. Et en dernière partie, vous
apprendrez à déboguer une application, vous confrontant ainsi à plusieurs types de bugs.

Vous trouverez dans ce cours :
- des méthodes à mettre en pratique lors de vos débogages pour mieux comprendre les erreurs ;
- des outils à utiliser pour vous faciliter la vie que vous pourrez réutiliser dans le monde professionnel ;
- de nombreux exemples et mises en situation pour vous permettre de pratiquer.

Le débogage est une compétence clé pour tous les développeurs et développeuses ! C’est d’ailleurs une compétence qui démarque les développeurs
des très bons développeurs. Vous n’avez donc aucune excuse pour ne pas la mettre en pratique !

:::info
**Objectifs pédagogiques**

À la fin de ce cours, vous serez capable de :
- Identifier le type de bug ;
- Déboguer du HTML et CSS ;
- Déboguer des bugs sur le DOM ;
- Déboguer des bugs sur l’API ;
- Déboguer une application.
:::
