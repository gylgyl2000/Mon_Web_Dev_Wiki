# Résumé du cours

## Résumé du cours

[Résumé du cours - Créez une application web avec Vue.js - OpenClassrooms](https://openclassrooms.com/fr/courses/6390311-creez-une-application-web-avec-vue-js/6916391-resume-du-cours)

Félicitations pour avoir terminé ce cours ambitieux !  Vous avez découvert tellement de concepts.

![](https://user.oc-static.com/upload/2020/05/20/15899604015526_Bannee-dimension-720-798-01.png)

Vous avez commencé à créer votre première application Vue.js sans outil de construction et découvert les principes fondamentaux des
applications Vue.js.

![](https://user.oc-static.com/upload/2020/05/20/15899604203211_Bannee-dimension-720-798-02.png)

Ensuite, vous avez fait évoluer votre application avec Vue CLI et configuré le routage de votre application.

![](https://user.oc-static.com/upload/2020/05/20/15899604277792_Bannee-dimension-720-798-03.png)

Puis, vous avez découvert comment créer une application avec une architecture basée sur des composants et comment gérer vos styles sous
différentes approches.

![](https://user.oc-static.com/upload/2020/05/20/15899604361318_Bannee-dimension-720-798-04.png)

Enfin, vous avez géré les données dans Vuex et découvert comment les mettre à jour et les modifier.

Avec ces connaissances fondamentales, vous êtes prêt à contribuer à la plupart des applications Vue.js ! En prime, vous êtes également prêt à
explorer de nouvelles technologies comme les applications Web progressives, le développement d'applications hybrides ou mobiles, et bien plus
encore. Et si vous souhaitez continuer, en fonction de votre compréhension de Vue, n'oubliez pas de consulter les ressources sur les tests,
l'animation, les bibliothèques de composants, et bien plus encore.

Quel que soit le chemin que vous suivrez, je sais que vous serez prêt à relever tous les défis qui vous attendent. J'espère que vous avez pris
autant de plaisir à suivre ce cours que j'en ai eu à le créer. Je vous souhaite bonne chance dans votre aventure en tant que développeur !
