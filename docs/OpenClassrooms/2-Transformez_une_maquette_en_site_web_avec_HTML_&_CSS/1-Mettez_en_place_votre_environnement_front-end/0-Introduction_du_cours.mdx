# Introduction du cours

[Mettez en place votre environnement front-end - OpenClassrooms](https://openclassrooms.com/fr/courses/6943241-mettez-en-place-votre-environnement-front-end)

<Vimeo vimeoId="491649859" />

<br />

En tant que développeur front-end, comme pour tout métier technique, il faut les bons outils.

Dans ce cours, je vais vous accompagner dans l’installation, la configuration et la découverte de trois outils essentiels :
- **Visual Studio Code**, qui sera votre outil principal pour écrire du code, avec son éditeur avancé, son explorateur de fichiers,
sa palette de commandes, et tout son outillage intégré ;
- **Chrome** et **Firefox**, deux navigateurs incontournables pour analyser et tester vos projets grâce à leurs puissants outils
de développeur, ou **_devtools_**.

_Prêt à vous lancer ? Rejoignez-moi dans ce cours !_

:::info
**Objectif pédagogique :**
À l'issue de ce cours, vous serez capable de mettre en place un environnement de développement front-end.
:::

:::note !
**Prérequis :**
Des connaissances de base en Git et en ligne de commande, que vous pouvez apprendre grâce à ces cours :
- [Apprenez à utiliser la ligne de commande dans un terminal](https://openclassrooms.com/fr/courses/6173491-apprenez-a-utiliser-la-ligne-de-commande-dans-un-terminal)
- [Utilisez Git et GitHub pour vos projets de développement](https://openclassrooms.com/fr/courses/5641721-utilisez-git-et-github-pour-vos-projets-de-developpement)

**Outils nécessaires** **:** 
- Visual Studio Code
- Google Chrome ou Firefox
- GitHub
:::
