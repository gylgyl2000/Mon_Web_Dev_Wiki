## Packages npm

### react
:::info

React est une bibliothèque JavaScript pour créer des interfaces utilisateur.

Le package React contient uniquement les fonctionnalités nécessaires pour définir les composants React. Il est
généralement utilisé avec un moteur de rendu React tel que `react-dom` pour le Web ou `react-native` pour les
environnements natifs.

**Remarque :** par défaut, React sera en mode développement. La version de développement inclut des avertissements
supplémentaires sur les erreurs courantes, tandis que la version de production inclut des optimisations de
performances supplémentaires et supprime tous les messages d'erreur. N'oubliez pas d'utiliser la version de
production lors du déploiement de votre application.

**Exemple d'utilisation**

`var React = require('react');`

- [react - npm](https://www.npmjs.com/package/react)
- [facebook/react: A declarative, efficient, and flexible JavaScript library for building user interfaces.](https://github.com/facebook/react)

:::

### react-dom
:::info

Ce package sert de point d'entrée au DOM et aux moteurs de rendu du serveur pour React. Il est destiné à être
associé au package générique React, qui est expédié comme `react` à npm.

- [react-dom - npm](https://www.npmjs.com/package/react-dom)

:::

### react-router
:::info

<img src="https://monwikidevweb.la-loutre.fr/img/outils/react-router.png" style={{ width: '80px', float: 'left', margin:'0 40px 20px 0'}} />

Routage déclaratif pour React.

**Remarque :** ce package fournit la fonctionnalité de routage de base pour React Router, mais vous ne voudrez
peut-être pas l'installer directement. Si vous écrivez une application qui s'exécutera dans le navigateur, vous
devriez plutôt installer `react-router-dom`. De même, si vous écrivez une application React Native, vous devez
plutôt installer `react-router-native`. Les deux installeront react-router en tant que dépendance.

- [react-router - npm](https://www.npmjs.com/package/react-router)
- [React Router: Declarative Routing for React.js](https://reactrouter.com/)
- [react-router/packages/react-router at master · ReactTraining/react-router](https://github.com/ReactTraining/react-router/tree/master/packages/react-router)

:::
