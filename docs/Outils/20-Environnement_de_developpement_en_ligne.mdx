### JSFiddle

:::info
<img src="https://monwikidevweb.la-loutre.fr/img/outils/jsfiddle.png" style={{ width: '80px', float: 'left', margin:'0 40px 20px 0'}} />

https://jsfiddle.net/

JSFiddle est un service IDE en ligne et une communauté en ligne pour tester et présenter
des extraits de code HTML, CSS et JavaScript créés par les utilisateurs et collaboratifs,
appelés 'fiddles'. Il permet des appels AJAX simulés. En 2019, JSFiddle a été classé deuxième
IDE en ligne le plus populaire par l'indice PopularitY of Programming Language (PYPL) en
fonction du nombre de recherches effectuées, directement derrière Cloud9 IDE, dans le monde
et aux États-Unis.

https://docs.jsfiddle.net/getting-started
:::

### Glitch

:::info
<img src="https://monwikidevweb.la-loutre.fr/img/outils/glitch.svg" style={{ width: '80px', float: 'left', margin:'0 40px 20px 0'}} />

https://glitch.com/

L'application Web Glitch a été lancée au printemps 2017 pour créer des applications Web
simples à l'aide de JavaScript. Bien que JavaScript soit le seul langage pris en charge,
d'autres langages peuvent être utilisés de manière non officielle. Présenté comme un outil
de « visualisation de la source » qui permet aux utilisateurs de « recombiner le code de
manière utile ». Glitch est un IDE en ligne pour JavaScript et Node.js avec et inclut
l'hébergement instantané, le déploiement automatisé et l'aide en direct des membres de la
communauté. Les fonctionnalités de l'IDE comprennent l'édition en direct, l'hébergement,
le partage, la gestion automatique des versions de la source et l'intégration Git. Glitch
se concentre sur le fait d'être une communauté amicale et accessible ; depuis son lancement,
plus d'un million de personnes ont utilisé le site pour créer des applications Web. Le site
Glitch est auto-hébergé (à l'exception de l'éditeur et de l'API), permettant aux utilisateurs
de visualiser ou de remixer le code source du site.
:::
