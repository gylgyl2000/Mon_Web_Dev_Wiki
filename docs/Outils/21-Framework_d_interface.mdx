# Framework d'interface

-----

### Bootstrap

:::info
<img src="https://monwikidevweb.la-loutre.fr/img/outils/bootstrap_logo.png" style={{ width: '80px', float: 'left', margin:'0 40px 20px 0'}} />

https://getbootstrap.com/

Bootstrap est une collection d'outils utiles à la création du design (graphisme, animation
et interactions avec la page dans le navigateur, etc.) de sites et d'applications web.
C'est un ensemble qui contient des codes HTML et CSS, des formulaires, boutons, outils de
navigation et autres éléments interactifs, ainsi que des extensions JavaScript en option.
C'est l'un des projets les plus populaires sur la plate-forme de gestion de développement
GitHub.
:::

### Google Fonts

:::info
<img src="https://monwikidevweb.la-loutre.fr/img/outils/Google_Fonts_logo.svg" style={{ width: '300px', float: 'left', margin:'0 20px 0px 0'}} />
Google Fonts (anciennement Google Web Fonts) est une bibliothèque de plus d'un millier de familles de polices gratuites et open source, un
répertoire Web interactif pour parcourir la bibliothèque et des API pour utiliser les polices via CSS et Android. Le site Web est disponible
dans plus de 135 langues. Le catalogue de polices Google est destiné à permettre la découverte et l'exploration des polices, et le service
est largement utilisé avec plus de 48 000 milliards de vues de polices au total. Les polices populaires incluent Roboto, Open Sans, Lato,
Oswald, Montserrat, Source Sans Pro et Raleway.

https://fonts.google.com/

:::
