# Somme des carrés(n)

[Training on Square(n) Sum | Codewars](https://www.codewars.com/kata/515e271a311df0350d00000f/train/javascript)

:::info Instructions
Complétez la fonction `squareSum` de manière à ce qu'elle élève au carré chaque nombre qui y est passé, puis additionne les résultats.

Par exemple, pour `[1, 2, 2]`, il devrait renvoyer `9` car `1^2 + 2^2 + 2^2 = 9`.
:::

```javascript
function squareSum(numbers){
  let sum = 0;
  for (let num of numbers) {
    sum += num * num;
  }
  return sum;
}
```
